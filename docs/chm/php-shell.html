PHP Shell Console

Console-style pseudo-terminal over HTTP for remote command execution through sftp.php.

Features

  • Prompt with host and working directory.
  • Command history (Up/Down).
  • Basic completion and list suggestions.
  • Colored output: prompt/command/output separation.

Open behavior

The Shell... button validates PHP auth first. On auth errors, console does not open and an explicit error is shown.

Known limits

  • This is not a full PTY/TTY emulator.
  • Some terminal-native behaviors may differ from PuTTY/OpenSSH shell.
