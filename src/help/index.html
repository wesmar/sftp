Feature Highlights

SSH/SFTP/SCP

  • Full SFTP protocol support with resume capability
  • SCP transfer mode (including SCP-only for restrictive hosts)
  • Mixed mode: SFTP listing + SCP transfers
  • Native PPK key support (v2 and v3 with Argon2)
  • Pageant integration for key management
  • Proxy support: HTTP CONNECT, SOCKS4a, SOCKS5
  • SSH Jump Host / ProxyJump through bastion servers

Shell Fallback

  • Works when SFTP/SCP are blocked by server policy
  • Multiple methods: raw cat, dd blocks, base64 encoding
  • Automatic resume with proper alignment
  • Command escaping for special characters
  • Compatible with minimal shell environments

PHP Agent (HTTP)

  • No SSH account needed - works with web hosting
  • Secure HMAC-SHA256 authentication
  • Full file operations: CRUD + rename + hash
  • Chunked upload with resume support
  • Auto chunk from PROBE recommendation (safe clamp + fallback)
  • Capability probing before operations
  • Shell execution via HTTP (when enabled)

Character Encoding

  • UTF-8 auto-detection for international filenames
  • Multiple codepage support (1250-1258, Asian, etc.)
  • Text/binary/auto transfer modes
  • Line ending conversion (CRLF/LF)
  • UTF-8 BOM preservation

LAN Pair & Jump Host

  • LAN Pair (SMB-like) for direct Windows-to-Windows pairing
  • Role-based pairing: Auto / Receiver / Donor
  • Per-session LAN Pair timeout with persistent INI storage
  • Jump Host / ProxyJump using SSH direct-tcpip without external ssh.exe
  • Separate fingerprint and authentication state for jump host and target host

Session Management & Import

  • Unlimited saved sessions
  • Encrypted password storage (Windows DPAPI)
  • Import from PuTTY (registry + Portable), WinSCP (registry + .ini), and KiTTY Portable
  • Checkbox picker — select any combination of sessions in one step
  • 4-path import memory — last 4 browse paths remembered per install
  • Native PPK conversion (v2 and v3/Argon2, no external tools)
  • Proxy settings per session

Localization

  • 15 languages: EN, PL, DE, FR, ES, IT, RU, CS, HU, NL, PT-BR, RO, SK, UK, JA, ZH-CN
  • External .lng files loaded at runtime — no rebuild needed
  • Auto-detected from Total Commander wincmd.ini
  • Dynamic dialog layout adjusts for label length in all languages

Quick Reference

Transfer Mode Selection

  1. Start with SSH (SFTP/SCP) - works for most servers
  2. If SFTP fails: try Use SCP for data transfers
  3. If SCP fails: try Use SCP for everything
  4. If still failing: enable Shell transfer fallback
  5. No SSH access: use PHP Agent (HTTP)
  6. Need bastion access to SSH target: use Jump Host
  7. Need direct Windows-to-Windows local transfer: use LAN Pair

Authentication Selection

  1. Have SSH key? Use Private key file (PEM or PPK)
  2. Using Pageant? Check Use SSH agent
  3. Only password? Enter in Password field
  4. Hosting without SSH? Use PHP Agent mode

Common Ports

  • SSH/SFTP: 22 (default), often custom (2222, etc.)
  • HTTP: 80
  • HTTPS: 443
  • SOCKS proxy: 1080
  • HTTP proxy: 8080
Practical tip: If the host blocks SFTP/SCP, this plugin can still transfer files via shell command pipeline (cat/dd/base64) where typical tools often fail.

About This Plugin

SFTPplug is a Total Commander filesystem plugin providing secure file transfer over SSH (SFTP, SCP), HTTP (PHP Agent), and direct local pairing (LAN Pair). It features comprehensive authentication support, Jump Host / ProxyJump for bastion-based access, shell fallback for restrictive hosts, PHP-based agent mode for web hosting environments, session import from PuTTY / WinSCP / KiTTY Portable, and runtime localization in 15 languages (PL/DE/FR/ES/IT/RU/CS/HU/NL/PT-BR/RO/SK/UK/JA/ZH-CN).
