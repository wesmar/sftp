Plugin Configuration (sftpplug.ini)

Plugin-wide settings that are not part of individual sessions are stored in the [Configuration] section of sftpplug.ini.

File location

Total Commander stores plugin ini files in its configuration directory, typically:

C:\Users\YourName\AppData\Roaming\GHISLER\sftpplug.ini

The exact path depends on how TC was installed:

TC installation typeTypical sftpplug.ini location
Standard install (per-user)C:\Users\YourName\AppData\Roaming\GHISLER\sftpplug.ini
Standard install (all users)C:\ProgramData\GHISLER\sftpplug.ini
Portable / USB install<TC folder>\sftpplug.ini
Tip: In Total Commander, open Configuration → Options → General — the path shown next to wincmd.ini is the TC configuration directory. sftpplug.ini is in the same folder.

If sftpplug.ini does not exist yet, the plugin copies a template (sftpplug.tpl) from the plugin directory on first run. You can also create the file manually.

Configuration keys

KeyDefaultDescription
Language (follows TC) Overrides the plugin UI language. See Language override below.

Language override

By default the plugin detects the language from Total Commander's own setting (wincmd.ini → LanguageIni). The Language= key overrides this detection.

Use it in two situations:

  • TC uses an unsupported language — the plugin supports 15 languages. If TC is set to a language not in that list (e.g. Finnish, Turkish, Arabic, Norwegian, Hebrew), the plugin falls back to English. Use Language= to pick the closest available language instead.
  • Wrong language detected — if TC uses a custom or community language file whose filename the plugin cannot identify (e.g. the plugin shows Chinese characters instead of your language), force the correct language here.

Example — force English: Open sftpplug.ini in a text editor, find or create the [Configuration] section, and add the Language= line: 
[Configuration]
Language=English
Restart Total Commander for the change to take effect.

Custom translation file

If your language is not in the list above, you can create a custom .lng file:

  1. Find the plugin directory — it contains the plugin DLL (sftpplug.wfx / sftpplug.wfx64) and a language\ subfolder.
  2. Copy one of the existing .lng files from the language\ subfolder as a starting point, for example:
    copy language\deu.lng → language\fin.lng
  3. Open language\fin.lng in a UTF-8 text editor (e.g. Notepad++). Each line has the format: 
    # This is a comment (ignored)
12345=Translated string here
12346=Another string
    Translate the strings on the right side of each =. Leave the numeric IDs on the left unchanged. Lines starting with # are comments.
  4. Save the file as UTF-8 (with or without BOM — both are accepted).
  5. Add to sftpplug.ini: 
    [Configuration]
Language=fin
  6. Restart Total Commander.
Partial translations are fine. Any string ID not present in the .lng file will fall back to English automatically. You only need to translate the strings you care about.
