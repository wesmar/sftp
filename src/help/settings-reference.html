Settings Reference (Complete Dialog Guide)

Main Connection Dialog

The main dialog is accessed via Net Connect (Ctrl+F1) in Total Commander. It contains all connection and transfer settings.

Connection Section

  • Connect to:
    • For SSH/SFTP: Enter hostname (e.g., example.com) or IP address
    • For PHP Agent/Shell: Enter full URL (e.g., https://example.com/sftp.php)
    • Supports custom ports: host:2222 or https://host:8443/sftp.php
    • IPv6 support: [::1]:22
  • Session: Dropdown with saved sessions. Use F7 to create new, F8/Delete to remove.
  • User name: SSH username (for SSH modes); optional label in PHP modes (not used for PHP auth)
  • Password:
    • SSH password authentication
    • PHP Agent HMAC authentication secret
    • Stored encrypted via Windows DPAPI when "Save password" is used

Authentication Section

  • Use SSH agent (Pageant):
    • Automatically tries keys loaded in Pageant
    • Non-intrusive: falls back to other methods if no suitable key
    • Shows detailed log of which keys were tried
  • Public key file: Optional path to public key (.pub, .pem)
  • Private key file:
    • Accepts PEM format (OpenSSH)
    • Accepts PPK format (PuTTY) — native conversion, no external tool or DLL needed
    • Supports encrypted keys with passphrase prompt
    • PPK v2 and PPK v3 (Argon2) both supported natively — Argon2 is built into the plugin

Transfer Mode Section

  • Transfer mode dropdown:
    • SSH (SFTP/SCP) - Standard SSH file transfer
    • Use SCP for data transfers (checkbox) - SFTP for listing, SCP for transfers
    • Use SCP for everything (no SFTP) (checkbox) - Pure SCP mode
    • Use shell transfer (for blocked hosts) (checkbox) - Shell command pipeline
    • PHP Agent (HTTP) - HTTP-based file operations
    • PHP Shell (HTTP) - HTTP pseudo-terminal
    • LAN Pair (SMB-like) - direct Windows-to-Windows pairing
  • Compress data with ZLIB: Enable SSH compression (may help on slow links)
  • Use shell transfer (for blocked hosts):
    • Enables cat/dd/base64 pipeline fallback
    • Use when SCP/SFTP are blocked by server policy
    • Supports resume with base64 alignment (3-byte boundaries)

Jump Host

  • Use jump host: Enables ProxyJump / bastion routing for SSH mode
  • Jump... Opens dedicated Jump Host configuration dialog
  • Jump Host settings are only active for SSH (SFTP/SCP)
  • Jump Host is disabled for PHP Agent, PHP Shell, and LAN Pair modes

LAN Pair Controls

  • Role: Auto / Receiver / Donor
  • Peer: Detected LAN peer dropdown
  • Session timeout: Minutes; 0 means no limit
  • Pair... Starts pairing or role negotiation

Advanced Options

  • Detailed log: Enable verbose logging; output appears in the TC log window (F5 log pane or separate log window)
  • Show entire directory listings in log: Includes full remote ls -la output in the log — useful for diagnosing SCP/shell listing problems

File Permissions (SFTP mode)

In SFTP transfer mode you can change file and directory permissions directly from Total Commander:

  • Use Properties (Alt+Enter) on a remote file or directory to view and edit its Unix permissions
  • Permission changes use SFTP setstat — the plugin reads current permissions first, applies your change, and writes back
  • Works on files and directories; symlinks are not followed
  • Not available in SCP-only, Shell, PHP Agent, or LAN Pair modes

Character Encoding

  • UTF-8 for filenames:
    • Auto-detect: Plugin tries to detect server encoding
    • Force UTF-8: Always use UTF-8 encoding
    • Force ANSI: Use system codepage
  • Text file types: Wildcard patterns for text mode transfers (e.g., *.txt;*.log;*.ini)
  • Transfer mode:
    • Binary: Raw byte transfer (default)
    • ASCII: Convert line endings (CRLF ? LF)
    • Auto: Use wildcard patterns to decide

Proxy Configuration Dialog

Accessed via Proxy... button. Configure network proxy for SSH connections.

Proxy Types

  • No proxy: Direct connection
  • HTTP CONNECT: HTTP proxy using CONNECT method
  • SOCKS4a: SOCKS version 4a (supports domain names)
  • SOCKS5: SOCKS version 5 (full authentication support)

Proxy Settings

  • Proxy server: Hostname or IP, optional port (proxy:1080)
  • Username: Proxy authentication username
  • Password: Proxy authentication password (stored encrypted)

Proxy Error Messages

  • Connection not allowed - Proxy policy blocks destination
  • Network unreachable - Target network not reachable via proxy
  • Host unreachable - Specific host not reachable
  • Connection refused - Target actively refused connection
  • TTL expired - Routing loop or timeout
  • Command not supported - Proxy doesn't support requested operation
  • Address type not supported - IPv6/hostname not supported by proxy

Keyboard Shortcuts (Connection Dialog)

  • F7 - Create new session (opens "New folder" dialog for naming)
  • F8 / Delete - Delete selected session
  • Enter - Connect with current settings
  • Alt+Enter - Edit selected session properties
  • Ctrl+Down - Open session dropdown

Session Storage

  • Sessions stored in sftpplug.ini in Total Commander directory
  • Passwords encrypted with Windows DPAPI (user-specific)
  • Import from PuTTY/WinSCP registry available via Import... button
Tip: After importing from WinSCP, verify transfer mode settings. SCP sessions are imported with scponly=1 and scpfordata=1 flags.
